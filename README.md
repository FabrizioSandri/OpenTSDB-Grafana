### OpenTSDB - Grafana
Unione Tra openTSDB e Grafana attraverso docker-compose

##### Esecuzione
per eseguire basta eseguire il seguente comando all' interno della directory contentent il file docker-compose.yml:
```
docker-compose up 
```
